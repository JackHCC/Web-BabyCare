# Web-BabyCare
BabyCare官网模型（ BabyCare official website model）

[项目开源地址](https://github.com/babycareorg)

[设计](jackhcc.github.io/web-babycare)

### 针对婴儿远程监护的物联网系统设计
+ Arduino开发版实现硬件端监控
+ Android实现手机软件端交互
+ 3D建模设计

[视频功能演示](https://www.bilibili.com/video/BV1pE411w71e/)
